# Bon Voyage ✈️

Minimalist travel itinerary planner with interactive mapping.

## Features
- **Interactive Map**: Visualize routes in real-time.
- **Itinerary**: Search, reorder (drag & drop), and manage destinations.
- **Flexible**: Toggle between **One Way** and **Round Trip**.
- **Transport**: Choose from various transport modes.
- **Local Save**: Automatically persists to your browser.

## Getting Started
```bash
npm install
npm run dev
```

## Tech Stack
Next.js, Leaflet, @hello-pangea/dnd, Lucide React.
